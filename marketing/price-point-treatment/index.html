Price points can provide a great deal of information in a condensed area, so it is important to use type and colour appropriately for legibility and impact.

This diagram outlines how the positioning and size of all elements are based on the height of the pricepoint numbers. The table below outlines the standard sizes to build upon. Price point cannot be bigger than the Bell logo on the piece.

All elements:

  • Size/positioning based on price-point number height (3x)
  • Left alignment
  • Type tracking 0
