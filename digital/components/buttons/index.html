Buttons trigger an action or an event. They let users know what will happen next.

Buttons

This is the default form and shape of a button.

  • Inside padding of 15 px between elements
  • Max. copy length: 18 characters
  • If label exceeds 18 characters, prefer a link with chevron, instead of a button.

Button variants

Button implementation example

Do
  • Respect max copy length of 18 characters.
  • Distinguish primary and secondary CTA's by alternate button styles.
Don't
  • When a CTA has more than 18 characters, it should switch to a hyperlink.
  • Clearly distinguish primary and secondary CTA's.

Cancel link

Cancel can be in the form of a link when the main action we want the user to focus on is the primary button.

Exception: Cancel can be a button when this is a primary call to action (CTA)
