Standard navigation

Base navigation for Bell websites. Up to 4 levels of navigation with respective background colors. Components and number of sections may vary depending on the platform.

  • Minimum and maximum margin respects layout grid
  • Inside padding of 30 px or 15 px between elements
  • Gray top navigation = #2D2E33
  • Main navigation = #00549a
  • Second level navigation = #002F71
  • Third level navigation = #F4F4F4
